## Quick Start

### Option 1: Vision Server (New Projects)

Build APIs with observability built-in:

```bash
npm install @getvision/server
```

```typescript
import { Vision } from '@getvision/server'
import { z } from 'zod'

const app = new Vision({
  service: { name: 'My API' }
})

app.service('users')
  .endpoint('GET', '/users/:id', {
    input: z.object({ id: z.string() }),
    output: z.object({ id: z.string(), name: z.string() })
  }, async ({ id }, c) => {
    // Tracing built into context
    const user = c.span('db.select', { 'db.table': 'users' }, () => {
      return { id, name: 'John' }
    })
    return user
  })

app.start(3000)
```

### Option 2: Vision Adapters (Existing Apps)

Add Vision to your existing Hono/Fastify app:

```bash
npm install @getvision/adapter-hono
```

```typescript
import { Hono } from 'hono'
import { visionAdapter, enableAutoDiscovery } from '@getvision/adapter-hono'

const app = new Hono()

// Add Vision observability
app.use('*', visionAdapter())
enableAutoDiscovery(app)

// Your existing routes
app.get('/users/:id', async (c) => {
  return c.json({ id: '123', name: 'John' })
})
```
